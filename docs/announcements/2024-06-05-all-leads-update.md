# Обновление по сегменту `all_leads`

**Дата:** 2024-06-05  
**Канал:** #marketing-product-sync

Команда продукта обновила поведение сегмента `all_leads`: теперь в него попадают только незарегистрированные пользователи. Для рассылок по зарегистрированным пользователям требуется запрос на отдельный сегмент/флаг.

## Что изменилось
- Сегмент `all_leads` исключает зарегистрированных пользователей.
- Кампании для зарегистрированных должны запускаться через выделенный сегмент (настройка продуктовой командой).

## Что нужно сделать маркетингу
1. Использовать новый порядок из Playbook (см. раздел 2.5) для запросов кампаний по зарегистрированным пользователям.
2. При необходимости кампании создать тикет с пометкой `registered_campaign` и приложить целевую аудиторию, цель, черновик сообщения, метрики.
3. Отслеживать подтверждение Product Ops о дате запуска перед коммуникациями.

## Следующие шаги
- Product Ops проверит и актуализирует календарь коммуникаций.
- Маркетинг подтверждает получение обновления реакцией в канале.
